## Run
1. `$ git clone https://github.com/C2BoT/code_shot.git`
2. `$ pip install -r requirements.txt`
3. <b>Paste your token in the</b> [config.py](https://github.com/C2BoT/code_shot/blob/master/config.py) <b>file</b>
4. `python3 main.py`
