## Run
1. `$ git clone https://github.com/hoosnick/picodebot.git`
2. `$ pip install -r requirements.txt`
3. <b>Paste your token in the</b> [config.py](https://github.com/hoosnick/picodebot/blob/master/config.py) <b>file</b>
4. `python3 main.py`
